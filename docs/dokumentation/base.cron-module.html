update_wps_processes()

source code 

This method update the list of WPS processes provided by the WPS Server. It takes a describe_processes_url of each wps server saved in database, open it, and parses the xml file, that comes in request message.

It will be checked if any object present in database. If no, it will be saved, else it will be overwritten with the actual information provided by the WPS server.

Returns: NoneType
None
